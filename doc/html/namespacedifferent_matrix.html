◆ craeteMatrix()

Matrix * differentMatrix::craeteMatrix ( int m,
int n )

Allocation of memory for the Matrix structure.

Each line consists of n+1 characters, where n+1 character is equal to '\0'. Therefore, you can work with each line as with a regular string: char*.

Parameters
mThe number of rows
nThe number of columns
Returns
Matrix* the fields m, n correspond to the arguments, iterator = 0, all characters will have the state '\o'
Exceptions
std::logic_errorInvalid row or column value
std::bad_allocWhen memory is not allocated by the NEW operator

Strs * differentMatrix::createStrings ( )

Allocation of memory for the Strs structure.

Pointers to the strings themselves are equal to nullptr. DEFAULT_CAPACITY cells are allocated for an array of strings. Initially, the value of DEFAULT_CAPACITY is 8.

Returns
Strs* Pointer to strings structure, field size = 0
